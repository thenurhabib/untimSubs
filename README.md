##### untimSubs use multiple subdomain enumeration tools and find all subdomains and save it in particularly text files.

<br>

### install
```bash
git clone https://github.com/thenurhabib/untimsubs.git
cd untimsubs
chmod +x install.sh
./install.sh
```
